# CLI 猜拳遊戲
這是用 Python 製作的命令列猜拳遊戲，玩家可以跟電腦對戰，採取回合制，並進行計分比賽。

## 版本說明
☛ V1 (基礎版)
- 剪刀石頭布基本對戰
- 電腦隨機出拳
- 5分制計分
- 防呆設計

☛ V2 (重構版)
- 將勝負判定邏輯抽出為 judge() 函式
- 程式碼結構更清晰

## 遊戲說明
- 玩家輸入: 剪刀 / 石頭 / 布
- 電腦隨機出拳
- 勝負判定如下:  
    剪刀 > 布  
    石頭 > 剪刀  
    布 > 石頭  
- 平手則不計分
- 先達5分者獲勝
- 輸入「結束」可退出遊戲

## 功能特色
- 電腦隨機出拳
- 單局勝負判定系統
- 對局計分
- 防呆設計，防止用戶錯誤輸入
- 5分制結束遊戲
- 最終勝負判定

## 操作說明
執行遊戲後輸入:  
剪刀 \ 石頭 \ 布

或輸入:  
結束  

即可退出遊戲  

## 輸出範例
請輸入要出的拳: 石頭  
電腦出:  剪刀  
恭喜您贏了  
目前比分 您: 1 : 電腦: 0
